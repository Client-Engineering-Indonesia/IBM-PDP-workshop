## 04 - Manage data catalog

1. Click hamburger icon -> click Governance -> click Catalogs -> click All catalogs -> click Create new catalog button

<img width="1728" height="421" alt="image" src="https://github.com/user-attachments/assets/4c120159-0dfa-4d26-8951-1c0537ab27ed" />

2. Set Name to "YOURNAME - PDP Catalog" -> tick up Enforce data protection rules -> click Create button

<img width="1728" height="989" alt="image" src="https://github.com/user-attachments/assets/e8e561c8-03e4-4594-9ed3-543cff375342" />

3. Go to project that you have created in lab 0 -> go to Assets tab -> click Db2 Metadata Enrichment for PDP -> select all tables -> click Publish 

<img width="1403" height="496" alt="image" src="https://github.com/user-attachments/assets/74ff0388-1035-41b8-83ee-c0a91c16ed12" />

4. Select "YOURNAME - PDP Catalog" -> click Next button

<img width="1586" height="936" alt="image" src="https://github.com/user-attachments/assets/84571dbf-9267-4e9d-bf7a-4f5d4e8f5b12" />

5. Review the activity then click Publish button

<img width="1589" height="935" alt="image" src="https://github.com/user-attachments/assets/6f44037d-9123-4720-adc3-46b27b56e546" />

6. Go to your catalog and check if all assets have been imported

<img width="1728" height="843" alt="image" src="https://github.com/user-attachments/assets/fe0987c0-8732-4a83-ac90-91e9b9892206" />

7. Go to Access control tab -> click Add collaborators button -> select Add user -> type "user" in collaboratos field -> click Add button
